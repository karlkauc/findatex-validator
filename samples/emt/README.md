# EMT V4.3 example files

Generated by `tools/build_emt_samples.py`. One clean reference plus
two deliberately broken files. EMT cross-field rules are not yet
encoded (see `EmtRuleSet` TODO), so only mechanical rule families
are exercised here.

| File | What it demonstrates |
|------|----------------------|
| `01_clean.xlsx` | All 40 M-flagged fields populated. |
| `02_missing_mandatory.xlsx` | Blanks NUM=5, 11, 12, 15, 16 → PRESENCE/* errors. |
| `03_bad_formats.xlsx` | Invalid ISO 8601 datetime / date, non-ISO-4217 currency, non-Y/N reporting flag. |

Open via `mvn javafx:run` → EMT tab → *Browse…* or run `mvn -Dtest=EmtExampleSamplesTest test`.
