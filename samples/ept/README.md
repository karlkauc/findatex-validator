# EPT V2.1 example files

Generated by `tools/build_ept_samples.py`. One clean reference plus
two deliberately broken files. EPT cross-field rules are not yet
encoded (see `EptRuleSet` TODO), so only mechanical rule families
are exercised here.

| File | What it demonstrates |
|------|----------------------|
| `01_clean.xlsx` | All 66 M-flagged fields populated. |
| `02_missing_mandatory.xlsx` | Blanks NUM=4, 9, 16, 17, 18 → PRESENCE/* errors. |
| `03_bad_formats.xlsx` | Invalid ISO 8601 datetime / date, non-ISO-4217 currency, non-Y/N reporting flag. |

Open via `mvn javafx:run` → EPT tab → *Browse…* or run `mvn -Dtest=EptExampleSamplesTest test`.
